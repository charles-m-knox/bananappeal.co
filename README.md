Generate a new site named `blog`:

```bash
hugo new site main

```

```bash
cd main/themes
git clone https://github.com/themefisher/parsa-hugo.git
```

Pull from the example [config.toml](https://github.com/themefisher/parsa-hugo/blob/master/exampleSite/config.toml) and put it into `blog/config.toml`.
